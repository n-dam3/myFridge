# myFridge
## 시작 가이드
*1. rcmn/sql.txt를 참조해 db 생성*  
*2. rcmn/ingrds.csv, rcmn/recipes.csv를 ingrdients, recipes table에 import*  
*3. src/main/resources/application.yml에 jpa, smtp, oauth2 설정 추가*
